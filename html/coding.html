    Print Comparator
    Technical Details
  • Computer vision-backed print comparator - designed for state analysis
  • Preprocesses images, converting them to grayscale, setting image size, cropping, denoising, and normalizing contrast
  • Projects two impressions into the same coordinate space via feature-based homography (using AKAZE/SIFT + RANSAC)
  • Produces two visual analysis modes: Binary Difference Map, backed by per pixel arithmetic (pixel-wise absolute difference between the two impressions) and Channel Mapped Color Overlays, where the base image is mapped to red, the target image is mapped to blue, and the overlaps between them are neutralzied.


    Semantic Map of AIC Prints
    Technical Details
  • Using Art Institute of Chicago's API, loaded all their artworks, filtering for prints
  • Collected relevant metadata (artist name, title, origin, date, medium) into a single "semantic string."
  • Passed the string into the CLIP text encoder, which turns the string into a vector in ℝ⁵¹² (512 dimensional space!)
  • These vectors are then passed into UMAP, which projects their 512 dimensions into 2d. This isn't perfect; no way to visualize 512 dimensions on a computer screen! It preserves local relationships, but it discards global geometry.
  • Reduced to points in 2D semantic space, each embedded print is assigned a tile space. On the frontend, these are filled with corresponding image thumbnails (pulled from the Art Institute of Chicago on page load)
  • User can zoom in/out and click on an artwork for metadata (metadata popover only works when zoomed in).
  • Neatly abstracted, easily reusable
